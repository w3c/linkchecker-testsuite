This small test suite has been built to test the link checkers at W3C, but could be used for any tool used to either check links in HTML documents, spiders, or miscellaneous Web User-Agents.

This test suite is work in progress. Please submit any issue or idea to the public mailing-list public-qa-dev@w3.org
